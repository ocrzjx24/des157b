Celsius to Fahrenheit Conversion

This function converts a temperature from Celsius to Fahrenheit using the standard formula AKA multiply by 9/5 then add 32. I thought of this one because of all the calculations I had to do in my science classes, and thought it'd be a nice and quick way to test arrow functions.

Named Function

    function cToF(celsius) {
        return (celsius * 9/5) + 32;
    }

    //testing
    console.log(cToF(0));   // 32
    console.log(cToF(100)); // 212

Function Expression

    const cToF2 = function(celsius) {
        return (celsius * 9/5) + 32;
    };

    //testing
    console.log(cToF2(0));   // 32
    console.log(cToF2(100)); // 212

Arrow Function

    const cToF3 = (celsius) => {
        return (celsius * 9/5) + 32;
    };

    //testing
    console.log(cToF3(0));   // 32
    console.log(cToF3(100)); // 212
